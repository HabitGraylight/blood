# Blood Clocktower Online

当前活跃版本在 `newVersion/`。这是新的 React + Firebase 版本，后续开发和部署都以它为准。

旧版 Node/SSE 原型已经归档到 `oldVersion/`，只作为历史参考保留，不再作为默认入口。

## 使用新版

```bash
cd newVersion
npm install
npm run dev
```

## 目录

```text
newVersion/   当前版本
oldVersion/   旧版原型归档
```

本地密钥、`.env`、`node_modules/` 和旧版本地用户数据不会进入仓库。
